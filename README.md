# Data-Visualization-PJ

庄吓海老师数据可视化PJ，具体内容见pdf文件。主程序为GUI.py。操作系统为win7。python第三方库为vtk8.1.0。

我们主要运用了python中的vtk和pyqt5设计出一个交互界面。用户通过拖动控制点实现3D物体的形变，并且物体的形变数据可以保存为ffd文件。使用的算法是b样条函数和贝塞尔函数。
